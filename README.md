**B端、C端**

C端：Consumer（也可理解为Customer），通常为消费者、个人终端用户使用的客户端。如：微信、淘宝、网易云音乐等。 

B端：Business，通常为企业内部或商家使用的系统或平台。如：企业内部ERP管理系统、财务管理平台等。



**文件夹释义**

- blog-business-boot：后台管理系统后端

- blog-business-vue2：后台管理系统前端

- blog-consumer：前台用户
